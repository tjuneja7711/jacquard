Introduction


The Product Creator creates a 'punch card' from a number of input parameters: a design, a warp and weft sequenct, a loom, and weaves. You can start the Weave Editor from within the Product Creator to edit weaves. Integrated in the Product Creator is the Simulation Editor, a program that allows you to edit a simulation that was created in the Product Creator. The following picture illustrates the different parts of the Loom Editor main window:



The Main Window


The various commands of the View menu allow you to customize the main view as you wish. In addition dialog boxes and toolbars can be moved to new locations by drag and drop.

Five toolbars are available in the Weave Editor:

  1. The Standard Toolbar
  2. Weave Selector
  3. Information View
  4. Colour Palette
  5. Status bar
  6. Title bar

Viewing and Working Modes


The Product Creator has several viewing modes, which you can use, if desired, at the same time.
These are:

  • Design View
  • Weaves View
  • Special Weaves View
  • Card View
  • Simulation View
